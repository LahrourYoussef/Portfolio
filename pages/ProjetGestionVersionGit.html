Gestion de Versions avec Git

Dans le cadre de mon BTS SIO, j’ai eu l’opportunité de travailler sur un projet de gestion de versions avec Git. Cette expérience a été déterminante pour renforcer mes compétences en développement logiciel.

Apprentissage des Fondamentaux

Gestion des Versions

L’un des principaux apprentissages a été l’utilisation de Git pour créer des dépôts locaux et distants. J’ai compris comment suivre les modifications de fichiers et maintenir un historique clair des versions, ce qui est essentiel pour la traçabilité.

Utilisation des Branches

La gestion des branches m’a permis d’isoler le développement de nouvelles fonctionnalités. Cela facilite le travail collaboratif, car je peux travailler sans risquer de déstabiliser la branche principale, tout en intégrant facilement les modifications une fois terminées.

Compétences Développées

J’ai acquis une maîtrise des commandes Git, telles que git add, git commit et git merge. De plus, j’ai découvert l’importance de l’intégration continue grâce à des plateformes comme GitLab, qui facilitent le partage du code et la collaboration.

Conclusion

Ce projet m’a permis d’améliorer mes compétences techniques et de mieux comprendre les enjeux du travail en équipe. Je réalise maintenant que la maîtrise de Git est un atout incontournable pour tout développeur qui veut réussir dans le domaine du développement logiciel.
