Dunewave – Locaboard

Dune Wave : L'expérience e-commerce dédiée au surf. Un projet full-front conçu pour allier performance visuelle et gestion fluide de catalogue.

Contexte et Objectifs

Ce projet a été l'occasion de consolider nos fondamentaux en développement web. L'objectif principal était de concevoir une interface e-commerce complète, en passant de la maquette à l'intégration, tout en respectant les standards du HTML et du CSS . C'était aussi notre première immersion dans la collaboration technique via Git et la gestion de projet avec Kanban.

Technologies Utilisées

HTML CSS JavaScript

Fonctionnalités Clés

Catalogue de planches

Affichage des produits avec des filtres de recherche pour une navigation fluide.

Calendrier de réservation

Module de réservation intuitive pour une sélection précise des créneaux de location.

  • Panier et simulation de paiement :
    Tunnel d'achat optimisé avec simulation de paiement sécurisé.
  • Système d'authentification :
    Interface utilisateur dynamique permettant la gestion du profil et de l'historique.

    • Défis et Apprentissages

    Le principal défi de ce projet a été l'organisation du travail en équipe. Apprendre à se répartir les tâches efficacement et à communiquer pour avancer rapidement a été un apprentissage majeur. Ce fut une excellente introduction à la collaboration en environnement de développement.
