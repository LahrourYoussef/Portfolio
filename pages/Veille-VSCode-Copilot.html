Veille technologique — 1ʳᵉ année de BTS SIO (SLAM)

Sujet : VSCode et GitHub Copilot

Ma veille technologique a porté sur Visual Studio Code (VSCode) et GitHub Copilot, deux outils devenus incontournables pour les développeurs. J'ai étudié leur fonctionnement, leurs avantages et leur impact sur la productivité des entreprises.

VSCode est un éditeur de code gratuit, open-source et extensible, tandis que GitHub Copilot, un assistant d'intelligence artificielle, aide à générer du code automatiquement, à réduire les erreurs et à accélérer le développement.

Cette veille m'a permis de comprendre comment l'IA s'intègre dans le développement logiciel et comment ces outils transforment la façon de coder et de collaborer.
